HTML
¿What is HTML?
HTML es el lenguaje estándar o más utilizado para crear páginas web, este se encarga de cómo está distribuidos los elementos de la página web. HTML utiliza diferentes tipos de "etiquetas" para marcar distintps elementos dentro de la página, esto permite a las páginas interpretar y mostrar información de distintas maneras.


Head and Body tags
La etiqueta de "head" se centra específicamente en los metadatos de la página, estos metadatos son los que se centran en información acerca de la página y no se muestra directamente en esta, son instrucciones para la página y los motores de búsqueda de cómo manejar la página.


Basic Tags
Section Tag
La etiqueta 'section' define secciones temáticas en una página web, agrupa todo el contenido asociado a las secciones. Se utiliza principalmente para dividir una página en diferentes partes lógicas, normalmente incluyen encabezado.

Article Tag
La etiqueta 'article' define contenido independiente y reutilizable en una página web, por ejemplo, posts, noticias, foros, etc. Se utiliza para artículos de blog, comentarios, etc, normalmente se utiliza con un encabezado y con pie de página.

Aside Tag
La etiqueta 'aside' define contenido complementario, o relacionado indirectamente con el tema principal que se presenta, como citas, anuncios, etc. Estos se muestran en la partes laterales de la página, de ahí su nombre en inglés de 'aside'. Es un elemento semántico de HTML que es muy eficiente ya que no rompe el flujo del artículo.

Header Tag
La etiqueta 'header' define el título o cabecera de una página web, contiene elementos de introducción, logos, formularios, etc. Es un elemento semántico visible para los usuarios que se puede utilizar en distintos puntos de la página para marcar encabezados de diferentes secciones de esta.

Footer Tag
La etiqueta 'footer' degine el pie de página de diferentes elementos dentro de una página web, este contiene información como derechos de autor, contacto, etc. Normalmente se pone al final de la etiqueta de 'body' o de otras partes del documento como podrían ser las etiquetas 'article' o 'section'.

CSS
¿What is CSS?
CSS (Cascading Style Sheets) es un lenguaje de programación principalmente utilizado para diseñar y estilizar páginas web hechas con HTML y JavaScript. El término de 'Cascading', que es en cascada, se refiere a la jerarquía que utiliza el buscador para resolver conflictos cuando más de un estilo aplican para el mismo elemento.
¿What is the purpose of CSS?
El propósito de CSS es el controlar como se presenta y se distribuye los elementos y la información en una página web, separando el diseño visual de la página de su contenido estructural.
¿How do you apply CSS to an HTML?
There are 3 ways to apply CSS to an HTML:
  1. Una de las maneras de aplicar CSS a un HTML es utlizando un archivo aparte que sea .css y referenciarlo dentro del HTML en la etiqueta 'head'.
  2. Otra manera es utilizando la etiqueta 'style' en la cual dentro ya defines los estilos a utilizar más adelante en el archivo
  3. La tercer manera es implementar los estilos que quieras dentro de cada etiqueta individual, utilizando el atributo 'style' dentro de estas.
¿What is a selector? ¿What is a CSS rule?
Un selector de CSS es un patrón utilizado para 'encontrar' o enfocarse en elementos específicos de HTML en una página web para que puedas aplicarle reglas de estilo a ellos.

Una regla de CSS, también se le puede referir como uno de los bloques más importantes de una página de estilos, consiste de un bloque selector y un bloque de declaración, los cuales al juntarse le dan las instrucciones al navegador para como estilizar elementos específicos de HTML.
Explain the following terms:
Tag Selector
Un seleccionador de etiquetas hace lo que se espera de su nombre, se enfoca en todos los usos de una etiqueta específica en un archivo HTML.
Class Selector
Igual que el anterior este hace lo que dice su nombre, se puede definir clases en etiquetas específicas y luego este seleccionador solo utiliza las etiquetas con la clase específica.
Tag and Class Selector
Este seleccionador hace los dos anteriores pero a la misma vez.
ID Selector
Le puedes asignar a elementos espec[ificos IDs únicas y luego con el seleccionador estilizar únicamente a estos.
¿What is a Class?
Una clase es un identificador utilizado para agrupar varios elementos de HTML para que compartan las mismas reglas de estilización.
¿What is an ID?
Un ID es un identificador único utilizado para enfocarse y estilizar un elemento HTML único y específico.
¿How are they related to CSS?
Se utilizan para agarrar elementos específicos y aplicarles reglas de estilización a solo estos sin afectar el resto de la página.
